# HeadTrackPad
Simülasyon oyunlarında kullanılmak üzere basit bir kafa takibine bağlı tuşa tıklama uygulaması yaptım yapay zekayı kullanarak. Algoritmasını kendim oluşturdum. Flicker gibi sorunları çözmek için kafanın dönmesi iki bölge arasında sürekli git gel yapmasın diye deadzone eklemek gibi çözümler düşündüm.

I developed a simple head-tracking keypress application for simulation games using AI. I designed the algorithm myself. To solve issues like flickering, I came up with solutions such as adding a deadzone to prevent the head rotation from constantly bouncing back and forth between two zones.

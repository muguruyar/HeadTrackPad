# HeadTrackMouse
Simülasyon oyunlarında kullanılmak üzere basit bir kafa takibine bağlı mouse hareketi uygulaması yaptım yapay zekayı kullanarak. Algoritmasını kendim oluşturdum. Flicker gibi sorunları çözmek için kafanın dönmesi arttıkça sensivity'i logaritmik olarak azaltmak ve deadzone eklemek gibi çözümler düşündüm.

Using AI, I created a simple head tracking-based mouse movement application for simulation games. I designed the algorithm myself. To solve issues like flickering, I thought of solutions such as adding a deadzone and logarithmically decreasing the sensitivity as the head rotation increases
